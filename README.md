# Drumit

A lightweight `.drumtab` editor and drum notation visualizer.

## Run

```bash
bun run serve
```

Open `http://localhost:5173`.

## Supports

- `.drumtab` text format.
- Live parse diagnostics.
- PDF-style compact grid preview.
- Standard drum-staff preview.
- Sections, bars, repeats, core kit pieces, and common modifiers.
